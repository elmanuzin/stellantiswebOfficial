Instruir usuário a aguardar até aparecer mensagem para reconfigurar

Abrir chamado como a seguir:

Tipo de chamada: Consulta geral
Atendida pelo grupo: L_BRA_XXX_SERV_DESK_N1_EUS
Descrição resumida: SD - Reset MFA
DESCRIÇÃO: Usuário (a) entrou em contato informando que está solicitando a configuração do MFA para logar.
Informação repassada:
usuário entrou em contato solicitando a configuração do MFA (Autenticação de dois fatores) para logar.
O usuário relatou que o código do seu MFA está dando incorreto. Em seguida, foi solicitado o reset do MFA e o usuário foi orientado a aguardar para reconfigurá-lo.
Durante a verificação, foi confirmado se o usuário instalou o aplicativo correto e se foi possível identificar o motivo do erro.
Em seguida, o reset do MFA foi realizado para permitir a reconfiguração.

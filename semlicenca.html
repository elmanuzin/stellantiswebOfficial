Usuário sem licença


Geralmente, quando um usuário precisa de acesso a um determinado software ou sistema,
ele precisa de uma licença válida para usá-lo. Se a licença ainda não foi criada,
o usuário deve aguarda para criação desse licença.

Tipo de chamada: Consulta Geral
Atendida pelo grupo: L_BRA_XXX_SERV_DESK_N1_EUS
Descrição resumida: SD - Informação/Dúvida sobre licença
DESCRIÇÃO: O usuário entrou em contato informando que estava tendo problemas com a licença do sistema.
Foi verificado que a licença estava inativa.
Usuário foi instruido a aguardar criação da licença.
